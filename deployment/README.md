# Deployment Assets

This folder contains deployment-oriented assets and references for Google Cloud and containerized release workflows.

## Included

- `cloud-run-backend.md` - backend container deployment on Cloud Run
- `cloud-run-frontend.md` - frontend container deployment on Cloud Run
- `secrets-manager.md` - recommended secret handling model

## Notes

- This repository is prepared for deployment but intentionally does not auto-deploy.
- Use `docs/deployment-guide.md` for end-to-end release steps.
