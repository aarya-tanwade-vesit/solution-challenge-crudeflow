## 🔌 Connect Your Backend

After the first run:

1. Open `.env.local` in a text editor
2. Update these URLs:
   ```env
   NEXT_PUBLIC_API_URL=http://localhost:8000/api
   NEXT_PUBLIC_API_WS_URL=ws://localhost:8000/ws
   ```
3. Save and refresh browser (F5)
4. Start your backend service
5. The app will now connect to your backend

See `BACKEND_INTEGRATION.md` for API specs.

## 🆘 If START.bat Doesn't Work

### Try PowerShell Version Instead:

1. Right-click `START.ps1`
2. Select "Run with PowerShell"
3. If it asks about execution policy:
   ```powershell
   Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
   ```

### If Neither Works:

1. Open terminal in project folder
2. Type:
   ```bash
   pnpm install
   pnpm dev
   ```
3. Open http://localhost:3000
