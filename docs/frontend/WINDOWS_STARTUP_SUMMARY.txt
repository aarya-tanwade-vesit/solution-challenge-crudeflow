## Windows Startup Scripts - Complete Summary

**Everything you need to run the project on Windows in one click.**

### ✅ What START.bat Does (Automatic)

```
1. Check Node.js installed
   ↓
2. Install/verify pnpm
   ↓
3. Run pnpm install (downloads dependencies)
   ↓
4. Create .env.local (from template)
   ↓
5. Start dev server on http://localhost:3000
   ↓
6. Open browser automatically
```

**Total time:** 2-3 minutes first run, 10-15 seconds after that

### 🚀 Usage (3 Steps)

1. **Extract project ZIP**
   ```
   C:\Projects\crudeflow-frontend\
   ```

2. **Double-click `START.bat`**
   - A terminal window opens
   - Automated setup begins

3. **Wait for "Dev server ready"**
   - Browser opens at `http://localhost:3000`
   - You see the CrudeFlow landing page
   - Done!

### 📋 Prerequisites (Automatic Check)

The script checks for:
- ✅ Node.js v18+ (auto-stops if missing, tells you how to fix)
- ✅ pnpm (auto-installs if missing)
- ✅ npm dependencies (auto-installs)
- ✅ `.env.local` (auto-creates from template)

### 🛑 To Stop

Press `Ctrl + C` in the terminal
Type `Y` and press Enter

### 🔧 If Node.js Not Found

The script will tell you:
1. Download from: https://nodejs.org/
2. Run installer
3. Check ✅ "Add to PATH"
4. Restart terminal
5. Try START.bat again

### ⚙️ Environment Setup

After first run, `.env.local` is created automatically.

Edit it to add your backend URLs:
```env
NEXT_PUBLIC_API_URL=http://localhost:8000/api
NEXT_PUBLIC_API_WS_URL=ws://localhost:8000/ws
```

### ❓ Common Questions

**Q: Do I need to install anything before running?**
A: No. Just Node.js. The script checks and tells you if it's missing.

**Q: What if pnpm isn't installed?**
A: The script auto-installs it for you.

**Q: How do I stop the server?**
A: Press Ctrl+C in the terminal.

**Q: How do I start it again?**
A: Double-click START.bat again.

**Q: Can I run multiple instances?**
A: Yes, copy folder and change PORT in .env.local.

**Q: Does it work on my old Windows?**
A: If it's Windows XP SP3+, yes. Use START.bat (not PowerShell).
